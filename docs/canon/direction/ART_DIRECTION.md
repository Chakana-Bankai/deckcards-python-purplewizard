# ART DIRECTION

## Meta visual
El arte de Chakana debe sentirse como una mezcla entre iconografia ritual, fantasia ilustrada, interfaz holografica y memoria sagrada. La meta no es producir ruido abstracto, sino imagenes con foco, atmosfera y forma reconocible.

## Estructura obligatoria
Cada imagen importante debe contener:
1. **Subject**: figura, criatura, arma, simbolo o estructura dominante
2. **Action**: energia, gesto, ritual, impacto o tension visible
3. **Environment**: contexto espacial que explique donde existe ese sujeto
4. **Symbol**: geometria o emblema que conecte la pieza al universo

## Identidad por set
### Base
- violeta profundo
- oro ceremonial
- geometria chakana
- guerrero cosmico, guardian, oraculo

### Hiperborea
- blanco helado
- azul polar
- oro antiguo
- templos, cristales, guardianes, auroras, observatorios

### Arconte
- negro
- rojo profundo
- purpura corrupto
- arquitectura rota, entidades opresivas, vacio y runas malignas

## Retratos
- formato 512x512 o derivado de alta nitidez
- foco en rostro, capucha, mirada, emblema y energia
- estilo radio portrait / 2.5D
- evitar siluetas simples o magos geometricos sin caracter

## Hologramas
- partir de retrato base, no de placeholder
- scanlines, glow, leve interferencia y RGB controlado
- legible antes que ruidoso

## Regla
Curated-first cuando exista pieza maestra. Procedural como apoyo, no como excusa para aceptar arte debil.


## Estandar de Carta Premium por Capas
Cada arte de carta debe aspirar a una composicion por capas, no a ruido abstracto uniforme.

### Capas obligatorias
1. **Fondo base**
   - mar, selva, desierto ritual, templo, espacio, vacio, ruinas, glaciar, observatorio
2. **Sujeto principal**
   - personaje, criatura, guardian, arconte, simbolo vivo o estructura dominante
3. **Objeto o foco**
   - espada, codex, sello, reliquia, altar, gema, chakana, mascara, fragmento ritual
4. **Efectos y adornos**
   - runas, halos, particulas, grietas, aura, rayos, niebla, trazas de accion
5. **Integracion final / profundidad 2D-2.5D**
   - bloom contenido, niebla de distancia, siluetas secundarias, separacion de planos, lectura focal

### Regla de lectura
Cada arte debe responder claramente:
- donde ocurre
- quien o que domina la imagen
- cual es el objeto/simbolo central
- que energia o accion esta ocurriendo

### Variacion por rareza
- **Comun**: composicion clara, menos ornamento, lectura rapida
- **Infrecuente**: mas capas atmosfericas, mejor objeto focal
- **Rara**: mejor silueta, mejor contraste, mas profundidad
- **Legendaria**: puesta en escena premium, sujeto dominante, objeto ritual memorable, integracion de capas mas rica
