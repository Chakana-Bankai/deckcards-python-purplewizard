# CHAKANA MASTER GAME BIBLE 1.0

## Juego
CHAKANA : PURPLE WIZARD

## Genero
Roguelike deckbuilder espiritual / tactico.

## Experiencia buscada
El jugador debe sentir que cada run es una lectura del mundo: combate, mapa, lore, shop, codex, hologramas, audio y arte deben formar una sola experiencia, no modulos sueltos.

## Nucleo dramatico
Chakana atraviesa planos fracturados para mantener el equilibrio frente a la presion de los Arcontes. Hiperborea emerge como memoria perdida del mundo, no como agregado externo.

## Sistemas pilares
- combate por cartas
- progresion por mapa
- packs y recompensas con identidad
- codex como biblia viva
- arte y audio dirigidos por contexto

## Estado objetivo pre-1.0
- run completa jugable
- HUD claro
- sets visibles y coherentes
- retratos 2.5D legibles
- musica por contexto no repetitiva
- arquitectura preparada para futura extraccion a Chakana Engine

## Regla editorial
Cuando haya conflicto entre infraestructura, placeholder y direccion del universo, debe prevalecer la direccion del universo siempre que no rompa runtime ni QA.
